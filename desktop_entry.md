# 添加应用程序快捷方式


## 以`pycharm`为例

```
cd /usr/share/applications
sudo gedit PyCharm.desktop
```

```
[Desktop Entry]
Version=1.0
Name=PyCharm
Exec=/home/ubuntu/pycharm-2019.2.3/bin/pycharm.sh
Terminal=false
Icon=/home/ubuntu/pycharm-2019.2.3/bin/pycharm.png
Type=Application
Categories=Development
```
